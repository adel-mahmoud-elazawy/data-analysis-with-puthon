# Car Sales Data Analysis Project 🚗

## 📌 Project Overview
This project focuses on analyzing car sales data using **Python**.  
The main goal is to clean the dataset, perform Exploratory Data Analysis (EDA), and visualize key insights to understand market trends and car pricing factors.

## 🚀 Features & Workflow
* **Data Cleaning:** Handled missing values, corrected data types (e.g., converting dates and categorizing company names), and ensured consistency.  
* **Exploratory Analysis (EDA):** Investigated unique values, distributions, and relationships between car features.  
* **Data Visualization:** Created charts to represent sales trends and price correlations.  
* **Memory Optimization:** Optimized data storage by using proper categorical types to reduce memory usage.  

## 🛠 Tech Stack
* **Language:** Python 3.x  
* **Libraries:**  
  * `Pandas`: Data manipulation and analysis  
  * `NumPy`: Numerical operations  
  * `Matplotlib` & `Seaborn`: Data visualization  

## 📊 Dataset
The dataset used in this project is sourced from **Kaggle**.  
It contains detailed information about car sales, including:
* Car IDs & Models  
* Sales Dates  
* Body Styles  
* Pricing & Engine Specifications  

## 📉 Key Insights
* SUVs were the most sold body style, accounting for nearly 40% of sales.  
* There is a strong correlation between **Annual Income** and the **Price** of purchased cars.  
* Certain companies consistently priced higher due to engine specifications and body style preferences.
